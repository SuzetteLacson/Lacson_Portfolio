# Bio

Hi, I'm Suzette — a Computer Engineering student who likes turning manual, repetitive work into software that runs on its own. My focus right now is QA automation: I'm interning at Pick.A.Roo, where I write automated tests for a real production mobile app.

Outside of testing, I build things — Arduino robots, AI-powered tools, and small web platforms — and lately I've been getting into mobile app development. If it can be automated, scripted, or soldered, I'm probably interested in it.
